## 产品设计草稿

### 新层次结构
```
首页
├── 册数选择（第一册、第二册、第三册...）
│   └── 课程列表
│       ├── 第1课
│       ├── 第2课
│       └── ...
│           ├── 学习模式（看字听音认读）
│           ├── 规划模式（预习/复习计划）
│           ├── 考核模式（测试认字）
│           └── 错题本（记录错误）
└── 设置
```

### 功能定义
| 模块 | 功能 |
|------|------|
| 学习 | 展示汉字，听发音，反复练习认读 |
| 规划 | 设定学习计划，复习提醒 |
| 考核 | 测试认字正确率，记录成绩 |
| 错题 | 错误汉字集合，重点复习 |
